# 💻 로컬 PC vs AWS 클라우드 비교

## 🏆 로컬 PC 운영 (추천!)

### ✅ 장점
```
1. 비용 0원 (완전 무료!)
2. 이미 있는 컴퓨터 활용
3. 데이터 보안 (외부 유출 없음)
4. 설정 간단 (Python만 설치)
```

### ⚠️ 주의사항
```
1. 장 시간(09:00~15:30) PC 켜두기
2. 인터넷 연결 안정적이어야 함
3. 정전 대비 (UPS 권장, 선택사항)
```

### 📋 필요 사양
```
CPU: i5 이상 (AI 추론용)
RAM: 8GB 이상
저장공간: 50GB (VI 데이터 저장)
OS: Windows/Linux/Mac 모두 가능
인터넷: 100Mbps 이상 권장
```

### 🚀 실행 방법
```bash
# 매일 아침 장 시작 전 실행
cd /home/user1/auto_trading
source learning/trading_bot/bin/activate

# 자동 매매 시작 (백그라운드)
nohup python trading_engine/realtime_trader.py > logs/trading.log 2>&1 &

# 장 마감 후 자동 종료 (선택)
# 또는 계속 켜두고 다음날 아침까지
```

### 💡 전기세
```
PC 소비전력: 약 100W
하루 6.5시간 (장 시간)
한 달 전기세: 약 3,000원

→ 거의 무시할 수준!
```

## ☁️ AWS 클라우드 (선택사항)

### ✅ 장점
```
1. 24시간 자동 운영 (외출 중에도)
2. 서버 다운 걱정 없음
3. 확장성 좋음 (종목 늘려도 OK)
4. 백업/모니터링 자동화
```

### ❌ 단점
```
1. 비용 발생 (월 50,000원)
2. 설정 복잡 (AWS 지식 필요)
3. 보안 설정 필요
```

### 💰 AWS 비용 상세
```
EC2 t3.small (2 vCPU, 2GB RAM):
  - 시간당: 약 40원
  - 월간 (730시간): 29,200원
  
EBS 스토리지 (50GB):
  - 월간: 5,000원
  
데이터 전송:
  - 월간: 3,000원
  
CloudWatch 로깅:
  - 월간: 2,000원

--------------------------------------
합계: 약 40,000~50,000원/월
```

### 🤔 언제 AWS 쓸까?
```
✅ 이런 경우 AWS 추천:
  - 외출/출장/여행 잦음
  - PC 24시간 켜두기 부담
  - 서버 다운 걱정 없이 안정적 운영
  - 비용 부담 가능 (월 5만원)

❌ 이런 경우 로컬 추천:
  - 집/사무실에서 주로 거래
  - 장 시간만 PC 켜둘 수 있음
  - 비용 절감 최우선
  - 설정 간단한 게 좋음
```

## ✅ FAQ

### Q: 로컬 PC 사양이 낮으면?
```
A: 최소 사양
   - CPU: i3 이상
   - RAM: 4GB 이상
   - 저장공간: 20GB
   
   → 중고 노트북으로도 충분!
   → 10만원대 중고 PC로 가능
```

### Q: 노트북으로 가능?
```
A: ✅ 완전 가능!
   - 장 시간만 열어두면 됨
   - 배터리 충전 상태 유지
   - 화면 끄고 백그라운드 실행 OK
```

### Q: 라즈베리파이로도 가능?
```
A: ⚠️ AI 모델 추론은 느림
   - 단순 전략: OK
   - Transformer AI: 권장하지 않음
   - 최소 i3 CPU 이상 권장
```

### Q: 집에 PC 없으면?
```
A: 옵션
   1. 중고 PC 구매 (10~20만원)
      → 1달 수익으로 충분히 회수
   
   2. AWS 바로 사용 (월 5만원)
      → 초기 투자 없음
   
   3. 친구/가족 PC 빌리기
      → 장 시간만 사용
```

### Q: 인터넷 끊기면?
```
A: 로컬 운영 시:
   - 재연결 시 자동 재시작
   - 누락된 데이터 보정
   - 큰 문제 없음
   
   AWS 운영 시:
   - 99.9% 가용성 보장
   - 걱정 불필요
```
