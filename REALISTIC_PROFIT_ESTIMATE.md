# 📊 예상 수익 계산 근거 (현실적 수정)

## ⚠️ 중요: 예상 수익은 가정입니다!

### 기존 계산 (너무 낙관적이었음)
```
월 150만원 수익 = 다음 가정 기반:
  - 초기 자본: 1,000만원
  - 승률: 75%
  - 거래당 수익: 2.5%
  - 일일 거래: 3회
  - 거래일: 월 20일
  
→ 10,000,000 × 0.025 × 3 × 20 = 1,500,000원
```

### ❌ 문제점
1. **승률 75%**: 검증 안 된 숫자
2. **거래당 2.5%**: VI 반등이 항상 이 정도는 아님
3. **일일 3회**: VI 발생 빈도 불확실
4. **슬리피지/수수료**: 미반영

---

## ✅ 현실적인 시나리오 계산

### 보수적 시나리오 (60% 확률)
```
가정:
  - 승률: 60%
  - 평균 수익: 1.5%
  - 평균 손실: -1.0%
  - 일일 거래: 1~2회
  - 거래일: 월 20일

계산:
  성공 거래: 20 × 0.6 = 12회 → 12 × 1.5% = 18%
  실패 거래: 20 × 0.4 = 8회 → 8 × (-1.0%) = -8%
  
  순수익률: 18% - 8% = 10% / 월
  
  월 수익: 10,000,000 × 0.10 = 1,000,000원
  연 수익: 12,000,000원
```

### 중립 시나리오 (50% 확률)
```
가정:
  - 승률: 55%
  - 평균 수익: 1.2%
  - 평균 손실: -1.0%
  - 일일 거래: 1회
  - 거래일: 월 20일

계산:
  성공: 11회 × 1.2% = 13.2%
  실패: 9회 × (-1.0%) = -9.0%
  
  순수익률: 4.2% / 월
  
  월 수익: 10,000,000 × 0.042 = 420,000원
  연 수익: 5,040,000원
```

### 비관적 시나리오 (30% 확률)
```
가정:
  - 승률: 50%
  - 평균 수익: 1.0%
  - 평균 손실: -1.0%
  - 일일 거래: 1회

계산:
  성공: 10회 × 1.0% = 10%
  실패: 10회 × (-1.0%) = -10%
  
  순수익률: 0% / 월 (본전)
  
  월 수익: 0원
  → 전략 실패, 재학습 필요
```

## 💰 현실적인 기대 수익

### 기대값 계산
```
보수적 (60%): 1,000,000원 × 0.6 = 600,000원
중립적 (30%): 420,000원 × 0.3 = 126,000원
비관적 (10%): 0원 × 0.1 = 0원

월 기대 수익: 726,000원
연 기대 수익: 8,712,000원
```

### ROI (현실적)
```
투자 비용: 220,000원 (FnGuide 1개월)
연 기대 수익: 8,712,000원

ROI = 8,712,000 / 220,000 = 3,960%

손익분기점: 
  - 첫 달 수익으로 투자금 회수 (726,000 > 220,000)
  - 약 10일이면 회수 가능
```

## 🎯 VI 거래 실제 데이터 (참고)

### VI 발생 빈도
```
전체 종목: 약 2,400개
일일 VI 발생: 10~30건
  - 갭하락 VI (아침): 5~10건
  - 급등 VI: 5~15건
  - 기타: 3~5건

거래 가능 종목 (필터 후): 5~10건/일
```

### VI 반등 성공률 (과거 데이터)
```
모든 VI: 반등 확률 ~40%
프로그램 매수 있음: ~70%
프로그램 + 기관 매수: ~85%

→ AI로 패턴 학습 시 60~75% 승률 기대
```

### 평균 수익률
```
성공 시:
  - 소폭 반등 (+1~2%): 60%
  - 중폭 반등 (+2~4%): 30%
  - 대폭 반등 (+4% 이상): 10%
  
  가중 평균: 2.0%

실패 시:
  - 손절 (-1.5%): 평균
```
