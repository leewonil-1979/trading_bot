```
"VI 이후 눌렸다가 [프로그램 매수가 들어오면] 반등하고,
 아니면 쭉 미끄러지는 것 같아"

→ VI 직후 순간적인 프로그램 매수세 학습 필수!
```

### QuantiWise로 가능한가?

```python
# QuantiWise 과거 데이터
program_trading_data = {
  'granularity': 'daily',  # 일봉만! ❌
  'timestamp': '2024-11-20',
  'program_net_buy': 5000000000  # 하루 전체 누적
}

# VI 전후 60초 분석 시도
vi_time = '2024-11-20 09:05:23'
program_flow_before = ???  # 데이터 없음 ❌
program_flow_after = ???   # 데이터 없음 ❌

→ "VI 직후 프로그램 매수 유입" 판단 불가능! ❌❌❌
```

### FnGuide로 가능한가?

```python
# FnGuide 과거 데이터
program_trading_data = {
  'granularity': 'second',  # 초단위! ✅
  'timestamp': '2024-11-20 09:05:23',
  'program_net_buy': 150000000
}

# VI 전후 60초 분석
vi_time = '2024-11-20 09:05:23'

program_flow_before = [
  09:04:23: -50M,
  09:04:24: -30M,
  ...
  09:05:22: -10M
]

program_flow_after = [
  09:05:24: +100M,  # 프로그램 매수 유입! ⭐
  09:05:25: +150M,
  09:05:26: +200M,
  ...
]

→ "VI 직후 프로그램 매수 유입" 정확히 포착! ✅✅✅
→ AI 학습 가능! ✅
```
